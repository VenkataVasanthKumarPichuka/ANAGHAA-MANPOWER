ANAGHAA-MANPOWER
Overview

ANAGHAA-MANPOWER is a modern and responsive manpower services website developed to showcase staffing, recruitment, workforce management, and related business services. The website provides an intuitive user interface that enables users to explore company information, services, and contact details efficiently.

The application is built using modern frontend technologies to ensure high performance, responsiveness, and maintainability.

Technologies Used:
React JS
Vite
Tailwind CSS
JavaScript (ES6+)
HTML5
CSS3


Features:
Responsive and modern user interface
Fast page loading with Vite
Mobile-friendly design
Service showcase sections
Contact and inquiry sections
Reusable React components
Clean and maintainable code structure
Prerequisites

Before running the project, ensure that the following software is installed on your system:

Node.js (Latest LTS Version Recommended)
npm (Node Package Manager)

Verify installation:

node -v
npm -v
Installation



Navigate to the project directory:

cd ANAGHAA-MANPOWER

Install project dependencies:

npm install

or

npm i
Running the Project

Start the development server:

npm run dev
